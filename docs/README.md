# Reference Documents

This directory contains reference materials for creating and verifying course syllabi.

## Contents

| File | Description |
|------|-------------|
| `cfd-syllabus-checklist.pdf` | LMU Center for Faculty Development syllabus checklist |

## Usage

### During Syllabus Creation
Consult the checklist while editing `index.html` to ensure all required elements are included.

### Before Publishing
Use the checklist to verify your syllabus meets institutional requirements before deploying.

## Adding Documents

Place additional reference materials (institutional policies, templates, guides) in this directory.
