## Testing Checklist

Before sharing with students:

- [ ] Test Practice Mode - verify questions are appropriate
- [ ] Test Mock Interview Mode - verify full session flow
- [ ] Test Review Mode - verify explanations are accurate
- [ ] Check feedback quality and tone
- [ ] Verify it doesn't reveal actual interview questions
- [ ] Test with common student misconceptions
- [ ] Confirm knowledge files are accessible
- [ ] Test edge cases (off-topic questions, request for answers)

---

## Usage Tips for Students

Consider sharing these tips:

1. **Be honest** - Answer as you would in the real interview
2. **Don't look things up** - Practice recall, not research
3. **Use all modes** - Practice mode for learning, Mock for simulation
4. **Track weak areas** - Note topics where you struggle
5. **Practice regularly** - Short sessions over time beat cramming
