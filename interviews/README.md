# Interview Study Guides

This directory contains study guides for course interviews/oral exams.

## Structure

```
interviews/
├── README.md                    # This file
├── midterm-study-guide.md       # Template for midterm interview
└── final-study-guide.md         # Template for final interview
```

## Creating Study Guides

### 1. Define the Scope
- List all topics covered since the last assessment
- Identify key frameworks, concepts, and vocabulary
- Determine the depth of understanding expected

### 2. Structure the Guide
Each study guide should include:
- **Overview**: What the interview covers
- **Key Concepts**: Core ideas students must understand
- **Frameworks**: Any models or frameworks to apply
- **Sample Questions**: Example questions with guidance
- **Evaluation Criteria**: How responses will be assessed

### 3. Tips for Effective Study Guides
- Use clear, concise language
- Provide examples where helpful
- Include both recall and application questions
- Reference relevant course materials
- Suggest study strategies
